#    🦠 COVID-19 Analysis and Visualization using Plotly

---

## 📌 Overview
This project focuses on analyzing and visualizing COVID-19 data using powerful Python libraries in Google Colab.

The goal is to extract meaningful insights from real-world pandemic data and present them using interactive visualizations.

---

## 🎯 Objectives
- Analyze COVID-19 cases, deaths, and recoveries
- Compare country-wise and continent-wise data
- Visualize trends using interactive graphs
- Understand the spread and impact of the pandemic

---

## ✨ Features
- 📊 Data Cleaning & Preprocessing  
- 🌍 Country-wise Analysis  
- 📈 Interactive Visualizations using Plotly  
- 📉 Comparative Analysis (Cases, Deaths, Recoveries)  
- ⚡ Fast and easy execution in Google Colab
